# NameNote 仕様書

## 概要

**NameNote** は、タッチペン・マウス・指で描けるデジタルノートアプリ。マンガのネーム作成を主な用途とする。

- **バージョン**: 1.0.0
- **対応環境**: Web ブラウザ（Chrome/Edge/Safari/Firefox）/ iOS / Android（Capacitor）
- **技術スタック**: React 19 + TypeScript + Canvas API + jsPDF + pdfjs-dist + Capacitor
- **UI言語**: 日本語

---

## 画面構成

```
┌─────────────────────────────────────────┐
│              ツールバー（下部）           │
├──────┬──────────────────────────────────┤
│      │                                  │
│ メモ │     ノート（見開きスプレッド）     │
│ 欄   │  ┌──────────┬──────────┐        │
│      │  │ 左ページ  │ 右ページ  │        │
│      │  └──────────┴──────────┘        │
└──────┴──────────────────────────────────┘
```

- メモ欄は左右どちらかに配置可能（設定で切り替え）
- ツールバーは画面下部に固定

## ツール

### ペン（キー: `P`）
- フリーハンド描画
- **サイズ**: 1〜30px（スライダー）、起動時デフォルト 3px
- **筆圧対応**: Apple Pencil などの圧力センサー付きスタイラスで線幅が変化
  - 圧力値 0.1〜1.0 に正規化、EMA（α=0.3）で平滑化
- **描画品質**:
  - 二次ベジェ曲線（quadratic curves）で滑らかな線
  - EMA プリスムーズ（α=0.6）で 60Hz サンプリングのギザギザを補間
  - HiDPI: 内部 Canvas を CANVAS_SCALE（通常2）倍で描画
  - coalesced events に対応（最大 32 イベント/フレームをバッチ処理）
- **クロスページ描画**: ページ境界をまたぐと両ページに自動分割して連続描画

### 消しゴム（キー: `E`）
- ピクセル消去（`destination-out` composite）
- **サイズ**: 1〜30px、起動時デフォルト 10px

### なげなわ選択（キー: `L`）
- 自由形の領域選択
- 選択後の操作: **コピー / 切り取り / 移動 / 貼り付け / 削除**
- ペースト時の操作:
  - **ドラッグ**: 位置移動（別ページへの移動も可）
  - **コーナードラッグ**: リサイズ（中心固定）
  - **上部ハンドルドラッグ**: 回転（0〜360°）
- 選択解除: Esc キー または別ツール選択

### テキスト（キー: `T`）
- キャンバス上をタップして文字を配置
- **フォント**: Hiragino Mincho ProN / 游明朝（明朝体、日本語対応）
- **フォントサイズ**: 10, 14, **18**, 24, 32, 48pt（デフォルト 18pt）
- **書き方向**: 横書き（horizontal-tb）/ 縦書き（vertical-rl）
- **行高**: フォントサイズ × 1.5
- 既存テキストをタップ → 編集ポップアップ（サイズ変更・書き方向変更・削除）
- テキストをドラッグ → 移動（メモ欄⇔ノートページ間の移動も可）
- 空文字列で閉じると自動削除

## 色

- **プリセット 11 色**: `#1a1a1a`（黒） / グレー系 4 色 / 白 / 赤 / オレンジ / 黄 / 緑 / 青 / 紫
- **カスタムカラー**: HTML5 color picker
- 色変更時はペンツールへ自動切り替え（テキストツール使用中を除く）

## 入力モード（ツールバー）

| モード | ペン / マウス | タッチ指 |
|---|---|---|
| **AUTO**（デフォルト） | 描画 | パン |
| **DRAW** | 描画 | 描画 |
| **PAN** | パン | パン |

- 2 本指以上のタッチは常にジェスチャー（ピンチズーム）として扱われる

## ノート構造

### スプレッド・ページ
- **スプレッド**: 左右 2 ページの見開き単位
- **ページサイズ**: 560 × 800 px（論理）
- **スプレッド幅**: 1128 px（560 + 8 + 560）
- **背景色**: クリーム色（`#fffef8`）
- スプレッド追加: ツールバーの ＋ ボタン

### 綴じ方向
| 方向 | 右ページ | 左ページ | ページめくり |
|---|---|---|---|
| **右綴じ**（デフォルト） | 奇数ページ（1, 3, 5…） | 偶数ページ（2, 4, 6…） | ◀ で次へ |
| **左綴じ** | 偶数ページ | 奇数ページ | ▶ で次へ |

### ページ一覧（概観）
- ツールバーの概観ボタンで開く
- サムネイル表示、ドラッグで並べ替え、ページ削除・挿入

### モバイル（幅 ≤ 700px）
- 1 ページずつ表示
- ナビゲーションボタンでページ単位移動

---

## メモ欄

- **サイズ**: 260 × 4000 px（縦長スクロール）
- **位置**: 左（デフォルト）/ 右（左利き対応）— ファイルメニューで切り替え
- **操作**: ペン・消しゴム・テキストツールすべて使用可能
- AUTO / PAN モードのタッチ指 → 縦スクロール
- テキストをメモ欄⇔ノートページ間でドラッグ移動可能

---

## ズーム・パン

- **ピンチズーム（2 本指）**: 0.3 倍〜3 倍。ピンチの中点が画面上で固定される
- **パン**: 1 本指ドラッグ（AUTO/PAN モード）または 2 本指ドラッグ
- zoom / pan はスプレッド切り替え時にリセット

---

## Undo / Redo

- **スタック**: スプレッド単位で独立、最大 **20** エントリ
- **記録対象**: Canvas 描画 + テキスト変更
- スプレッド切り替え時に現スプレッドの履歴を保存・復元

### キーボード（テキスト編集中を除く）

## 保存・書き出し

### 自動保存
- 操作後 **600ms** のデバウンスで localStorage に自動保存
- ツールバーの保存ドットで状態表示: グレー=保存済み / オレンジ=保存中 / 赤=未保存

### プロジェクトファイル（`.namenote`）
- **形式**: JSON
- **内容**: 全ページ PNG（Base64）+ テキスト配列 + 設定
- **ファイル名**: `namenote_{YYYY-MM-DDTHH-MM}.namenote`
- **動作**:
  - 初回: ファイルピッカーで場所を指定
  - 再保存: 同じファイルへ上書き
  - 名前を付けて保存: 常にピッカー表示

### 書き出し
| 形式 | 内容 | ファイル名 |
|---|---|---|
| **JPG（スプレッド）** | 左右ページ + テキスト合成、品質 0.92 | `namenote_p{N}-{N+1}.jpg` |
| **PDF（全ページ）** | A4 縦、1 ページ 1 枚、右→左の順 | `namenote_{YYYY-MM-DD}.pdf` |

### インポート
- `.namenote` ファイルを開いて復元
- **PDF インポート**: 全ページをスプレッドとして取り込み

## データ永続化（localStorage）

| キー | 型 | 内容 |
|---|---|---|
| `namenote_spread_count` | string（数字） | スプレッド総数 |
| `namenote_page_{i}_{L\|R}` | PNG DataURL | 各ページの Canvas データ |
| `namenote_memo` | PNG DataURL | メモ欄の Canvas データ |
| `namenote_text` | JSON 文字列 | TextObject[] 全テキスト |
| `namenote_binding` | `"left"` \| `"right"` | 綴じ方向 |
| `namenote_memo_position` | `"left"` \| `"right"` | メモ欄の位置 |

## ファイルメニュー項目一覧

| 項目 | 動作 |
|---|---|
| このスプレッドをJPG書き出し | 現在の見開きを JPEG 保存 |
| 全ページをPDF書き出し | 全スプレッドを PDF 保存 |
| 名前を付けて保存… | `.namenote` で新規保存 |
| プロジェクトを開く… | `.namenote` ファイルを読み込み |
| PDFを読み込む… | PDF を全ページ取り込み |
| 左綴じ / 右綴じに切り替え | 綴じ方向を変更 |
| メモを右 / 左に移動 | メモ欄の位置を切り替え |
| ノートを初期化… | 全データ削除（確認ダイアログあり） |
